# Flask 实践
> 参考文档 [flask-tutorial](https://www.geeksforgeeks.org/python/flask-tutorial/)

## 构建Python环境
安装virtualenv， 用于创建独立Python项目虚拟环境
```shell
$ pip install virtualenv 
```
创建Python虚拟环境
```shell
$ virtualenv venv 
```
激活虚拟环境
```shell
# Windows下使用该脚本激活
$ . venv\Scripts\active

# Widows 下使用Git bash命令行激活
$ source venv/Scripts/activate

# Linux环境下使用该脚本激活
$ source ./venv/bin/activate

```
## 安装Flask
```shell
$ pip install flask
```
